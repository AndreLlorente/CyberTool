This program was made in honor of the best teacher ATEC has ever had, Papi Romao. (thanks for everything, we're glad we got to learn from someone who showed so much interest and enthusiasm in every class! :) )

As for the program prerequisits:
- pip install speech_recognition
- pip install mysql-connector-python
- pip install OTXv2
- pip install re
- pip install psutil
- pip install pandas

After downloading all the dependencies, you should extract the zip file to a folder and run the file named CYBERTOOL.py (main program that will import all the functions and tools)

Keep in mind:
This program was designed for Linux machines only.
This program has the option to connect with mariaDB so that you can extract the data directly to a database, if you want this function to work properly you will need to update the following lines of code in the following files:
- processlogs.py - line 50 and 51 (update the mariaDB user and password)
		 - line 31 (replace whats inside the variable log_files with the file path to the auth.log 			    file)

- processlogshttphttps.py - line 73 and 74 (update the mariaDB user and password)
- vtdatabase.py - line 21 and 22(update the mariaDB user and password)
- exceltodb.py - line 31 and 32(update the mariaDB user and password)

- Some functions will prompt the user for the location of the file they want to use for data extraction, we recommend that you move the files that you want to use to the same directory the program is running on.
